# Border-Force-Simulator

# controlls
w to speed up
s to slow down
a and s to turn

I to inspect a boat
h to mark it as safe
k to blow the boat up

# how to play
use the radar to find a boat, fly over it slow down and press 1. if the first 2 digits of its visa sum up to the last digit or 2 then its legit, otherwise its fake.
e.g. 5611 = safe because 5+6=11, 112 = safe because 1+1=2, 991 = fake because 9+9 doesnt = 1
